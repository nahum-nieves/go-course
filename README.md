# Go Course

## Required VS Code Extensions

To get the best development experience for this Go course, please install the following VS Code extensions:

### 1. Go
The official Go extension for VS Code provides rich language support including IntelliSense, debugging, code navigation, and more.

**Install:** Search for "Go" in VS Code Extensions marketplace or install via command:
```
ext install golang.go
```

### 2. Cobalt2
A beautiful theme for VS Code that provides excellent syntax highlighting and a pleasant coding environment.

**Install:** Search for "Cobalt2" in VS Code Extensions marketplace or install via command:
```
ext install wesbos.theme-cobalt2
```

### 3. Material Icon Theme
Provides Material Design icons for files and folders in the VS Code file explorer, making it easier to identify file types at a glance.

**Install:** Search for "Material Icon Theme" in VS Code Extensions marketplace or install via command:
```
ext install PKief.material-icon-theme
```

## Installation

You can install all extensions at once by opening the Command Palette (Cmd+Shift+P on macOS) and searching for "Extensions: Install Extensions", then search for each extension by name.
